Load 'Time' table into model 8

A Time table is useful in tracking reports at a granular level (eg. the hour or minute of an event). To use a Time table, ensure your data model has a transactional value that includes Time. 



Step 1: Click the Data Icon on the left of your screen, then click New Table on the top (See screenshot below)

Step 2: Copy and Paste the DAX Code below into the window, then click the  icon. (See GIF below)

Time = 
VAR _series =
    GENERATESERIES(1, 1440, 1)
VAR _time =
    ADDCOLUMNS( _series, "TimeandDay", TIME( 0, [Value], 0))
RETURN
    ADDCOLUMNS(
        _time,
        "TimeKey", FORMAT([TimeandDay], "hhmm"),
        "Actual Time", FORMAT([TimeandDay], "HH:MM AM/PM"),
        "Hour", HOUR([TimeandDay]), 
        "Hour Extended", FORMAT([TimeandDay], "H AM/PM"),
        "Minute", MINUTE([TimeandDay]),
        "AMPM", FORMAT([TimeandDay], "AMPM")
    )
